## 功能分析

### 1. 核心功能
此插件是一个**解锁Pro/VIP功能**的越狱插件，主要功能包括：

#### 加密/解密功能
- `BEncryptString:` - 字符串加密
- `BDecryptString:` - 字符串解密
- 使用Base64编码

#### 清理功能
- `cleanCache` - 清理缓存
- `clearUserDefaults` - 清理UserDefaults
- `clearKeyChain` - 清理钥匙串
- `clearAppDatas` - 清理应用数据

#### Keychain操作
- 使用自定义类 `llyKeychain` 进行钥匙串操作
- 支持密码存储、读取、删除
- 支持账户管理

### 2. Hook技术
插件使用了CydiaSubstrate框架进行方法Hook：
- `_MSHookFunction` - C函数Hook
- `_MSHookMessageEx` - Objective-C方法Hook

## 转换为Loon插件的思路

### 方法1: 使用Loon的Script功能

Loon支持使用JavaScript脚本进行请求/响应修改，可以通过以下方式实现类似功能：

#### 步骤1: 分析应用的网络请求
1. 使用抓包工具（如Charles/Fiddler）捕获应用的网络请求
2. 查找与VIP/Pro状态相关的API请求
3. 分析请求头、请求体、响应体

#### 步骤2: 编写Loon脚本
```javascript
// Loon Script示例
// 拦截VIP状态检查请求

let url = $request.url;
let body = $response.body;

// 修改响应体中的VIP状态
if (url.includes("/api/user/info")) {
    let obj = JSON.parse(body);
    obj.data.isVip = true;
    obj.data.vipExpireTime = "2099-12-31";
    obj.data.isPro = true;
    body = JSON.stringify(obj);
}

$done({body});
```

## 实际操作建议

### 方案A: 抓包+脚本修改（推荐）
这是最简单的方法，不需要深度逆向：

1. **抓包分析**
   - 安装Charles证书到iOS设备
   - 配置SSL代理
   - 使用应用并捕获所有请求

2. **识别关键接口**
   - 查找包含 `vip`, `pro`, `premium`, `subscribe` 的请求
   - 分析响应JSON结构

3. **编写Loon脚本**
   - 使用 `$response.body` 获取响应
   - 修改JSON中的关键字段
   - 返回修改后的响应

### 方案B: Frida Hook（高级）
如果需要更深入的分析：

1. **安装Frida**
   ```bash
   pip install frida-tools
   ```

2. **编写Hook脚本**
   ```javascript
   // hook.js
   if (ObjC.available) {
       // Hook所有返回bool的方法
       for (var className in ObjC.classes) {
           if (ObjC.classes.hasOwnProperty(className)) {
               var methods = ObjC.classes[className].$ownMethods;
               methods.forEach(function(method) {
                   if (method.includes('isVip') || 
                       method.includes('isPro') || 
                       method.includes('isPremium')) {
                       console.log('[+] Found: ' + className + ' ' + method);
                   }
               });
           }
       }
   }
   ```

3. **运行Hook**
   ```bash
   frida -U -f com.abishkking.maleworkout -l hook.js
   ```

### 方案C: 静态分析（最复杂）
使用IDA Pro/Ghidra进行静态分析：

1. **加载dylib到IDA Pro**
2. **查找字符串引用**
   - 搜索 "vip", "pro", "premium"
   - 查看交叉引用

3. **分析关键函数**
   - 查看函数调用图
   - 理解逻辑流程

4. **还原算法**
   - 如果有加密算法，尝试还原
   - 分析验证逻辑

let method = $request.method;
let headers = $request.headers;

console.log('Intercepted: ' + url);

try {
    if (body) {
        let obj = JSON.parse(body);
        
        if (obj.data) {
            obj.data.isPremium = true;
            obj.data.isPro = true;
            obj.data.isVip = true;

    }

}

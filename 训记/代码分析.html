💡 解密原理

r()函数实际上是b()函数的别名

function b(opt_attributes, locals) { // 1. 获取加密字符串表 var params = a(); // 2. 计算索引 replacementHash = replacementHash - 216; // 3. 获取加密数据 var data = params[replacementHash]; // 4. RC4解密 data = b["pMTJwy"](data, key); // 5. 返回解密后的字符串 return data; }

RC4解密算法

代码使用了RC4流加密算法，密钥是传入的第二个参数（如"f8gZ"、"vKED"等）

🎯 PRO解锁机制

步骤 操作 目的
1 拦截API请求 获取原始请求
2 修改请求头 添加PRO用户标识
3 修改User-Agent 伪装成特定版本
4 发送修改后的请求 欺骗服务器
5 服务器返回PRO数据 成功解锁

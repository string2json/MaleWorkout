## 🔧 PRO解锁机制

### 解锁原理（基于推测）

```
1. 拦截API请求: /whole_user_info_v4
2. 修改请求头:
   - Authorization: Bearer PRO_TOKEN
   - User-Agent: trainnote/1 CFNetwork/1325.0.1 Darwin/21.1.0
3. 欺骗服务器返回PRO用户数据
4. 应用显示PRO状态
```

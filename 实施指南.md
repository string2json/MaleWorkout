# 逆向工程实施指南

## 快速开始（推荐方法）

### 方法1: 使用Loon插件（最简单）

#### 步骤1: 安装Loon
1. 在App Store购买并安装Loon
2. 打开Loon，完成初始配置

#### 步骤2: 导入插件
1. 将 `MaleWorkout_Pro.plugin` 和 `maleworkout.js` 上传到云盘或GitHub
2. 在Loon中：配置 → 插件 → 添加插件 → URL
3. 输入插件URL并下载

#### 步骤3: 启用MITM
1. 在Loon中：配置 → MITM → 添加主机名
2. 添加: `api.maleworkout.com`
3. 安装并信任Loon的CA证书

#### 步骤4: 测试
1. 打开MaleWorkout应用
2. 查看Loon日志，确认请求被拦截
3. 检查应用中的VIP/Pro功能是否解锁

## 方法2: 抓包分析（推荐用于学习）

### 准备工作
- Charles Proxy (macOS/Windows) 或 Fiddler (Windows)
- iOS设备
- SSL Kill Switch 2 (越狱设备)

### 步骤1: 配置Charles

#### macOS
```bash
# 1. 安装Charles证书
帮助 → SSL代理 → 安装Charles根证书

# 2. 配置SSL代理
代理 → SSL代理设置
勾选"启用SSL代理"
添加: *:443
```

#### Windows
```
1. Help → SSL Proxying → Install Charles Root Certificate
2. Proxy → SSL Proxying Settings
3. 勾选 Enable SSL Proxying
4. Add: *:443
```

### 步骤2: 配置iOS设备

#### 非越狱设备
```
1. 确保iOS设备和电脑在同一WiFi
2. Charles: 帮助 → 本地IP地址 (查看IP)
3. iOS: 设置 → WiFi → 配置代理 → 手动
   服务器: [电脑IP]
   端口: 8888
4. iOS: Safari访问 chls.pro/ssl
   下载并安装证书
5. iOS: 设置 → 通用 → 关于本机 → 证书信任设置
   启用Charles证书
```

#### 越狱设备
```
1. 安装 SSL Kill Switch 2 (Cydia)
2. 设置中启用 SSL Kill Switch 2
3. 按非越狱步骤配置代理
```

### 步骤3: 抓包分析

```
1. 打开MaleWorkout应用
2. 在Charles中查看所有请求
3. 筛选包含以下关键词的请求:
   - vip
   - premium
   - pro
   - subscribe
   - user
   - profile
   - account

4. 查看请求和响应内容
5. 记录关键接口URL和参数
```

### 步骤4: 分析响应结构

找到用户信息接口后，分析响应JSON：

```json
{
  "data": {
    "userId": "12345",
    "isVip": false,           // ← 关键字段
    "isPremium": false,       // ← 关键字段
    "vipExpireTime": null,
    "subscription": {
      "type": "free",
      "isActive": false       // ← 关键字段
    }
  }
}
```

### 步骤5: 编写修改脚本

根据实际响应结构修改 `maleworkout.js`：

```javascript
let body = $response.body;
let obj = JSON.parse(body);

// 根据实际结构修改
obj.data.isVip = true;
obj.data.isPremium = true;
obj.data.subscription.isActive = true;

body = JSON.stringify(obj);
$done({body});
```

## 方法3: Frida动态Hook（高级）

### 准备工作
- 越狱iOS设备
- Mac或Windows电脑
- Python环境

### 步骤1: 安装Frida

#### macOS/Linux
```bash
# 安装Frida工具
pip install frida-tools

# 验证安装
frida --version
```

#### Windows
```powershell
# 使用pip安装
pip install frida-tools

# 验证安装
frida --version
```

### 步骤2: 在iOS设备安装Frida

```
1. 打开Cydia
2. 添加源: https://build.frida.re
3. 搜索并安装 Frida
4. 重启设备
```

### 步骤3: 连接设备

```bash
# USB连接设备
# macOS需要安装usbmuxd
brew install usbmuxd

# 转发端口
iproxy 2222 22

# 列出设备
frida-ls-devices

# 列出设备上的进程
frida-ps -U

# 列出已安装的应用
frida-ps -Uai
```

### 步骤4: 运行Hook脚本

```bash
# 方法1: 附加到运行中的应用
frida -U -n "MaleWorkout" -l hook_vip.js

# 方法2: 启动应用并注入
frida -U -f com.abishkking.maleworkout -l hook_vip.js --no-pause

# 方法3: 保存输出到文件
frida -U -f com.abishkking.maleworkout -l hook_vip.js -o output.log
```

### 步骤5: 分析输出

查看脚本输出，找到所有相关方法：

```
[+] Found: UserProfile -isVip
[+] Found: SubscriptionManager - isPremium
[+] Found: PaymentValidator - hasValidSubscription
[Hook] UserProfile -isVip
  Original return: 0x0
  Modified to: true
```

### 步骤6: 精确Hook

根据输出结果，编写精确的Hook脚本：

```javascript
if (ObjC.available) {
    // Hook特定类的特定方法
    Interceptor.attach(
        ObjC.classes.UserProfile['- isVip'].implementation,
        {
            onLeave: function(retval) {
                retval.replace(0x1);
            }
        }
    );
    
    Interceptor.attach(
        ObjC.classes.SubscriptionManager['- isPremium'].implementation,
        {
            onLeave: function(retval) {
                retval.replace(0x1);
            }
        }
    );
}
```

## 常见问题解决

### 问题1: 抓包看不到HTTPS请求

**解决方案**:
```
1. 确认已安装并信任Charles证书
2. 越狱设备安装SSL Kill Switch 2
3. 检查Charles的SSL代理设置
4. 确认端口8888未被占用
```

### 问题2: Loon插件不生效

**解决方案**:
```
1. 检查MITM配置是否正确
2. 确认URL匹配规则
3. 查看Loon日志是否有错误
4. 检查脚本语法是否正确
```

### 问题3: Frida连接失败

**解决方案**:
```
1. 确认iOS设备已越狱并安装Frida
2. 检查USB连接
3. 重启frida-server: killall frida-server
4. 重新运行iproxy
5. 检查frida版本是否匹配
```

### 问题4: 应用闪退

**解决方案**:
```
1. 可能是签名验证失败
2. 尝试使用更温和的Hook方式
3. 检查是否有反调试机制
4. 使用substrate方式而不是直接修改
```

## 进阶技巧

### 1. 多重验证绕过
某些应用有多重验证：
- 本地验证（可Hook）
- 服务器验证（需修改响应）
- 签名验证（需破解）

### 2. 时间戳验证
```javascript
// 修改时间相关验证
obj.data.expireTime = "2099-12-31T23:59:59Z";
obj.data.expireTimestamp = 4102444799;
```

### 3. 加密数据解密
如果响应数据是加密的：
1. 找到解密函数
2. Hook解密后的数据
3. 或者在解密函数处修改返回值

### 4. 证书绑定绕过
某些应用使用证书绑定：
```bash
# 使用 objection 工具
objection -g com.abishkking.maleworkout explore
android sslpinning disable
```
